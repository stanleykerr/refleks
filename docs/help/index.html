Help & Documentation

Get started with RefleK's, troubleshoot common issues, and learn how to make the most of your aim training analytics.

Getting Started

Installation

  1. Download the latest version from the GitHub releases page
  2. Run refleks.exe
  3. RefleK's will automatically detect your KovaaK's stats folder and start tracking your practice sessions.

Note: On first run, RefleK's creates two folders on your system:
1) ~/.refleks — stores app settings and local traces.
2) %AppData%/refleks-x.y.z — Windows roaming data used by the app's renderer (x.y.z is the app version).

First-Time Setup

RefleK's is designed to work out of the box with minimal configuration:

  • Stats folder: Automatically detected from your KovaaK's installation
  • Sessions: Runs are grouped with a 15-minute gap (adjustable in settings)
  • Mouse tracking: Disabled by default (enable in settings if desired)

Features Guide

How does automatic tracking work?

RefleK's monitors your KovaaK's stats folder for new CSV files. When you complete a scenario:

  • The new stats file is detected automatically
  • RefleK's parses the data and adds it to your history
  • The run appears in your Sessions and Scenarios pages
  • Benchmark progress updates if applicable
Understanding Sessions

Sessions group your scenarios based on timing:

  • Default gap: 15 minutes (customizable in settings)
  • Auto-grouping: Runs within the time gap are grouped together
  • Session stats: View aggregated performance for the entire session
  • Navigation: Quickly jump between scenarios in a session

Adjust the session gap in Settings → General → Session Gap

Tracking Benchmark Progress

RefleK's includes all major KovaaK's benchmarks:

  • Browse benchmark lists in the Benchmarks page
  • Your progress updates automatically as you play scenarios
  • Mark favorites for quick access

Steam Integration: RefleK's uses your Steam account to fetch official benchmark rankings. This is detected automatically on Windows.

Mouse Movement Recording

Enable mouse tracking to record and analyze your mouse paths:

  • Enable in settings: Settings → Advanced → Mouse Tracking
  • When active: Records during KovaaK's scenarios only
  • Playback: Replay with pause, scrub, and speed controls
  • Privacy: All recordings stored locally, never uploaded
  • Performance: Minimal impact on system resources

Note: Mouse tracking is Windows-only due to platform-specific APIs.

Launching Scenarios from RefleK's

Click any scenario name in RefleK's to launch it directly in KovaaK's:

  • No need to copy/paste scenario names
  • Works from all pages
Customizing Themes

RefleK's includes several built-in themes to match your setup:

  • Go to Settings → General → Theme
  • Choose from Dark (default), Light, Nord, Dracula, and more
  • Theme changes apply instantly
  • Your preference is saved automatically

Troubleshooting

Stats folder not detected

If RefleK's can't find your KovaaK's stats folder:

  1. Go to Settings → General → Stats Folder
  2. Enter the path to your KovaaK's stats folder (usually C:\Program Files (x86)\Steam\steamapps\common\FPSAimTrainer\FPSAimTrainer\stats)
New runs not appearing

If new scenarios aren't showing up:

  • Verify the stats folder path in Settings is correct
  • Try restarting RefleK's to reset the file watcher
  • Ensure KovaaK's is writing stats files (check the stats folder)
Mouse tracking not working

If mouse tracking isn't recording:

  • Verify it's enabled in Settings → Advanced → Mouse Tracking
  • Make sure RefleK's is running while you play KovaaK's
  • Verify you're on Windows (mouse tracking is Windows-only)
Benchmark progress not updating

If your benchmark ranks aren't updating:

  • Ensure you have an internet connection
  • Verify Steam is installed and you're logged in
  • Check that the scenario name exactly matches the benchmark requirement
“steam ID not found” error on Benchmarks

RefleK's uses your Steam account to fetch official benchmark progress. On Windows, it detects your SteamID from Steam's loginusers.vdf. If detection fails, you'll see steam ID not found when opening a benchmark.

Fix it with one of the options below:

  1. Make sure Steam is running and you're logged in, then restart RefleK's.
  2. Set the correct Steam install directory in Settings → Advanced → Steam install directory (e.g. C:\Program Files (x86)\Steam).
  3. Or manually set your SteamID via Settings → Advanced → SteamID override. Paste your SteamID (looks like 84609298453789401).
  4. Advanced: You can also set the environment variable REFLEKS_STEAM_ID before launching the app.
